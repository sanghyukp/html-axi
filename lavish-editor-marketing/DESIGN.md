# Lavish Editor Marketing Video Design

## Style Prompt

Create a calm, premium product demo that feels like a late-evening reading room around a precise technical artifact.
The product surface is dark ink chrome, cream type, and one brass attention color.
The video should feel polished, restrained, and browser-native, with the artifact as the bright paper surface inside the room.

## Colors

- Ink canvas: `#0f1115`
- Panel ink: `#11141a`
- Top bar ink: `#171a21`
- Cream type: `#f7f3ea`
- Muted steel: `#b9c0cf`
- Brass accent: `#f4c95d`
- Agent sage: `#8fe39e`
- User amber surface: `#25230f`

## Typography

- Display and brand moments: `EB Garamond`
- Product UI and body: `Geist`
- Terminal and file paths: `Geist Mono`

## Motion

- Prefer fades, focus pulls, and small measured moves.
- Use brass only when the viewer needs to see the interaction target.
- Avoid bounce, elastic motion, glassmorphism, gradients, blue-purple accents, and generic SaaS copy.
